Dealer Warranty Planner

1. Unzip the project.
2. Open a terminal in the project folder.
3. Run npm install.
4. Run npm run dev.
5. Open http://localhost:3000.
6. To deploy on Vercel, upload this folder as a Next.js project.

This package includes a front-end login screen only. To make login real, connect it to Supabase, Firebase Auth, Clerk, Auth0, or your own backend.
